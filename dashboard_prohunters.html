Venda de Armas — 3 etapas obrigatórias
Consulta de estoque
  • Sempre usar a planilha de controle de armas — nunca o Bling.
1
Venda
Planilha → marcar vendida → nº série + peso → inserir no pedido → imprimir → bunker → Financeiro
2
Autorização de compra
Imprimir 1ª NF → conferir fisicamente → assinar → NF com nº série → enviar ao cliente (CRAF)
3
CRAF recebido
Imprimir última NF → conferir nº série → pesar → Trello → aguardar data de envio
REGRA DE OURO — NUNCA PULAR ETAPAS
Guia de Trânsito (GT)
GT — Ezequiel Transportadora (terrestre)
  • Modelo em: Dropbox › Comercial › Documentações Para Preencher › Guias e Minutas de Trânsito
  • Editar campos em vermelho: nº guia (últimos 3 dígitos aleatórios), nº NF, data, destinatário/CNPJ/endereço, SFPC do estado, produto, qtd (unidade), volume (caixa), marca, código de barras (gerador no diretório, fonte "sem texto"), validade (60 dias)
  • NUNCA salvar. Imprimir → carimbar (metade sobre o selo) → João ou Luis assinar → digitalizar
  • Salvar: Dropbox › Expediente › Pro Hunters › Órgãos Anuentes › EXÉRCITO › GT para Clientes › 2026
  • Nome: GT-2026-06-02 - Selo nº 11.542.630 - Nome do Cliente - Produto
GT — Cliente retirando pessoalmente
  • Transportador = dados do próprio cliente. Mesmo processo.
GT — Redespacho Gollog (aéreo)
  • Usar somente quando vendedor avisar que é Gollog
  • Preencher também Verso GT + Minuta eletrônica (Minuta Preencher)
  • Gollog cobra cliente diretamente · Taxa Ezequiel: cliente paga direto, solicitar comprovante e encaminhar pelo WhatsApp · Pagamento à vista
  • Impressão do verso: folha com escrita para cima e código de barras para trás
Portal Postal
Etiqueta — Envio normal
  • portalpostal.com.br › Etiqueta › Gerar etiqueta › PAC
  • NF no Bling: Vendas › NF de Saída › cliente › Imprimir NF simplificada
  • Nome/empresa/aos cuidados = cliente · Chave NF = número abaixo do código de barras · Conteúdo = descrição · Valor unitário e declarado = preço un. da NF
Logística Reversa
  • portalpostal.com.br › Logística Reversa › Gerar Autorização de Postagem (Reversa) · Serviço: PAC · Validade: 10 dias corridos
