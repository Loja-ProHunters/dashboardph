Guia de Trânsito (GT)
GT — Ezequiel (terrestre)
  • Modelo: Dropbox › Comercial › Documentações › Guias e Minutas
  • Editar em vermelho: nº guia, nº NF, data, destinatário, SFPC, produto, qtd (unidade), volume (caixa), marca, código de barras, validade 60 dias
  • NUNCA salvar. Imprimir › carimbar › assinar (João ou Luis) › digitalizar
  • Salvar em: Dropbox › Expediente › Pro Hunters › EXÉRCITO › GT para Clientes › 2026
  • Nome: GT-2026-06-02 - Selo nº 11.542.630 - Nome do Cliente - Produto
GT — Cliente retira pessoalmente
  • Transportador = dados do próprio cliente. Mesmo processo.
GT — Gollog (aéreo)
  • Usar somente quando vendedor avisar. Preencher Verso GT + Minuta eletrônica. Gollog cobra cliente direto.
